# 01 - Objectif du projet

Ce projet a pour but de concevoir une manette complète avec une démarche d’ingénierie réelle :
- Apprendre les bases de l’électronique et du prototypage
- Comprendre la gestion d’énergie (batterie, charge, régulation)
- Concevoir un firmware pour la communication Bluetooth et USB
- Créer un PCB sur mesure à partir d’un prototype
- Modéliser et imprimer un boîtier adapté
- Documenter chaque étape sur GitHub
